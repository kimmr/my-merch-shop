## General info
A shopping website where I can add, post, edit, delete items. 
	
## Technologies
Project is created with:
* Node.js
* Express.js
* mySQL with sequelize in "mySQL" branch
* MongoDB with mongoose in "master" branch
* Javascript
* HTML/CSS/Bootstrap
